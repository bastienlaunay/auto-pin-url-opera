Help

Add urls in the textbox above, one per line, to have them
automatically pin the tab when visiting the page.

If any part of the url contains one of the entries, it will
trigger the tab to become pinned.

For more advanced users, you may also use regular expressions
to match patterns in the url.

Examples :
google.com URLs containing `google.com`
[1-9] URLs containing digits
o{2,} URLs containing 2 or more `o` in a row.
